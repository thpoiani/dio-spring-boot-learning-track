== Introdução

Esta API permite o gerenciamento de tarefas, incluindo criação, consulta, atualização e remoção.

== Modelo de Dados

=== Task

|===
| Campo        | Tipo    | Descrição

| id           | String  | Identificador único da tarefa
| title        | String  | Título da tarefa
| description  | String  | Descrição detalhada
| status       | String  | Status da tarefa (ex: PENDING, COMPLETED)
|===

== Códigos de Status

|===
| Código | Descrição

| 200 OK       | Requisição executada com sucesso
| 201 Created  | Recurso criado com sucesso
| 204 No Content | Recurso removido com sucesso
| 400 Bad Request | Erro de validação
| 404 Not Found | Recurso não encontrado
|===

---

== Observações

- Todos os endpoints retornam JSON
- Datas devem estar no formato ISO-8601
- O campo `status` deve seguir os valores definidos no backend
